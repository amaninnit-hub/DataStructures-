# Data Structures 

A collection of fundamental data structures implemented from scratch in Java to build strong foundation in logic and problem solving. 

## Implemented So Far

### Linked List 
A custom single linked list built using Node objects connected by pointers. 

**Features** 
- 'addFirst()' - adds a node to the beginning of the list
- 'addLast()' - adds a node to the end of the list
- 'delete()' - removes a node by value, handling head, middle, and not-found cases
- 'print()' - travereses and displays the entire list

### Stack (LIFO) 
A custom stack built using the same Node structure, following Last In First Out behavior. 

**Features**
- 'push()' - adds an element to the top
- 'pop()' - removes and returns the top element
- 'print()' - disolays the stack from top to bottom

### Queue (FIFO) 
A custom queue built using Nodes, following First In First Out behaviour.

**Features**
- 'enqueue()' - adds an element to the back
- 'dequeue()' - removes and returns the front element
- 'print()' - displays the queue from the front and back 

## Concepts Used 
- Nodes and pointers
- Traversal using a 'current' reference
- LIFO vs FIFO behaviour
- Handling edge cases (empty structures, head/front removal, value not found)
- Object Oriented Programming
